Especificidade do CSS

O que é especificidade?

A especificidade é a maneira como o navegador decide qual regra CSS deve ser aplicada a um elemento quando há várias regras que se aplicam a ele.

Como a especificidade é calculada?

A especificidade é calculada com base nos seletores usados em uma regra CSS. Cada seletor tem um valor de especificidade associado a ele.

O valor de especificidade é calculado da seguinte maneira:

  • 1 ponto para cada seletor do tipo ou seletor universal (*).
  • 10 pontos para cada seletor de classe, pseudo-classe ou atributo.
  • 100 pontos para cada seletor de ID.
  • 1000 pontos para cada estilo inline.

Como o navegador decide qual regra aplicar?

Quando há várias regras que se aplicam a um elemento, o navegador escolhe a regra com a maior especificidade. Se houver empate, a regra que aparece por último no código é aplicada.

Exemplos de cálculo de especificidade

Veja alguns exemplos de como a especificidade é calculada:

  • h1 - Especificidade: 1-0-0-0
  • ul li - Especificidade: 0-0-0-2
  • .menu a.active - Especificidade: 0-1-2-0
  • #conteudo p - Especificidade: 0-1-0-1
  • style="color: red;" - Especificidade: 1-0-0-0

Observe que o valor de especificidade é sempre representado por quatro números separados por hífens. O primeiro número representa o valor de especificidade dos seletores de ID, o segundo representa o valor de especificidade dos seletores de classe, pseudo-classe e atributo, o terceiro representa o valor de especificidade dos seletores de tipo e o último representa o valor de especificidade dos seletores universais.
